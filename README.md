
# Documentation

> This is a Covid19 dashboard to view basic country-wise statistics.

### Features

- Numerical Statistics
- Statistics using Line graph and Bar graph
- Search over 150 Countries
- Interactive Graphs

### Technologies

- ReactJS
- Material UI
- React-Countup
- Chart.js
